## Getting Started

1. First of all, clone the repository.
2. Install your dependencies using
```bash
npm install
# or
yarn
```
3. Run the development server:
```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build on Local


1. First of all, clone the repository.
2. Install your dependencies using:
```bash
npm install
# or
yarn
```
3. Build the production:
```bash
npm run build
# or
yarn build
```
4. Running production in local: 
```bash
npm run start
# or
yarn start
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
